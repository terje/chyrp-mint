Mint for Chyrp

A module for Chyrp that enables Mint stats in Chyrp as well as RSS tracking with the Bird Feeder pepper.

Mint for Chyrp is a module for Chyrp that enables site statistics for your blog using Mint.

The module includes the Mint javascript in the header and enables RSS tracking in your blog feed using the Bird Feeder pepper. It is set up with the default mint and bird feeder paths but these can be changed in the Settings pane of Chyrp admin if you have a custom Mint installation.

Install

Download and unzip the mint folder in your modules folder inside your Chyrp installation.

Install the module by dragging and dropping it to the enable moduled in Chyrp admin.

NOTE: You should remove any references to the Mint javascript if you have made such references in your theme files already, this module will insert the required javascript automatically.
